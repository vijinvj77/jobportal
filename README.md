# Job Portal

A full-stack job management system built with:
- **Backend:** Spring Boot 3 (Java)
- **Frontend:** Angular 18
- **Database:** MySQL

## Structure
- `jobportal-backend/` — REST API (Spring Boot)
- `jobportal-frontend/` — Admin Panel (Angular)

## Getting Started
```bash
cd jobportal-backend
mvn spring-boot:run
